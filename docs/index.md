# Swarm Dashboard Documentation

Welcome to the Swarm Dashboard documentation!

## Overview

Swarm Dashboard is a real-time monitoring tool for AI agent swarms. It provides a beautiful web UI for tracking agent progress, viewing live activity feeds, and monitoring completion status.

## Features

- **Real-time Updates** - Dashboard auto-refreshes every 2 seconds
- **Smart Completion Detection** - Automatically detects when agents finish
- **Beautiful UI** - Capybara-inspired color palette with light/dark themes
- **Clickable Details** - View live activity, tools used, files created
- **Zero Dependencies** - Pure Python standard library

## Getting Started

```bash
pip install swarm-dashboard
swarm-dashboard serve
```

See the [Installation Guide](installation.md) for more options.
